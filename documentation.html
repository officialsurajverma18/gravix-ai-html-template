Introduction

Gravix AI is a next-generation AI SaaS template built with HTML5, CSS, and a custom Vanilla JS motion engine.

Folder Structure

style.css          # Global Design System (Variables)
app.js             # All interactions (Vanilla JS)
index.html         # AI SaaS Demo
crm.html           # CRM Dashboard Demo
documentation.html # This file

Theme Customization

Change the core colors in style.css using CSS variables.

:root {
  --color-primary: #6366f1; /* Indigo */
  --color-secondary: #8b5cf6; /* Violet */
  --bg-body: #0f172a; /* Background */
}

Animation Attributes

Use data attributes to trigger animations without writing JS.

Scroll Reveal

Add data-aos="fade-up" to any element.

3D Tilt

Add data-tilt to any card container.
